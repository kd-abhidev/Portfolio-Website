## Project Overview

Built a DevOps focused portfolio website to showcase projects, skills, and hands on work.

The site highlights cloud deployments, automation scripts, and system monitoring setups.

Designed with clean UI, responsive layout, and fast load time.

---

## Key Impact

* Showcases real DevOps projects in one place
* Improves recruiter visibility with clear project sections
* Provides direct access to GitHub work and resume
* Built responsive design for mobile and desktop
* Optimized for fast loading and simple navigation
